# Your Task

Two string variables have been provided for you: `name` and `favoriteColor`. Use `Console.WriteLine()` to:

- Print the name on one line.
- Print the favorite color on another line.

## Method Signature

```cs
public static void PrintNameAndColor()
```

## Expected Output

```cs
Alex
Blue
```

## Hints

- 💡 1. Use `Console.WriteLine()` and pass the variable name inside the parentheses
- 💡 2. Remember: when printing a variable, don't use quotes around the variable name - `Console.WriteLine(name)` prints the value, `Console.WriteLine("name")` prints the word "name"
- 💡 3. You need two separate `Console.WriteLine()` statements - one for each variable
