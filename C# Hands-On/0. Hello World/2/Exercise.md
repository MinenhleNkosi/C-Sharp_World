# Your Task

Write a method that:

- Reads the user's name using `Console.ReadLine()`
- Stores it in a string variable
- Prints the name using `Console.WriteLine()`

## Method Signature

```cs
public static void GreetUser()
```

## Expected Results

```cs
Input: Alice
Output: Alice
```

## Hints

💡 1. Use `Console.ReadLine()` to read what the user types.

💡 2. Store the result in a string variable like `string name = Console.ReadLine();`

💡 3. Use `Console.WriteLine()` to print the variable
