# Your Task

Complete the method that takes a number and returns its double (multiplied by 2). You need to:

- Create an `int` variable to store the result
- Multiply the `number` parameter by 2
- Use `return` to send the result back

## Method Signature

```cs
public static int DoubleNumber(int number)
```

## Expected Results

```cs
DoubleNumber(5) -> 10
DoubleNumber(3) -> 6
```

## Hints

💡 1. Declare an `int` variable to store your result using: `int result = ...`

💡 2. Use the multiplication operator `*` to multiply the number by 2: `number * 2`

💡 3. Use the `return` keyword followed by your variable name to send the result back, like: `return result;`
