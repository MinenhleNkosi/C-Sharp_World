# Your Task

Create a greeting by combining `Console.Write` and `Console.WriteLine`. First, use `Console.Write` to print `Hello, `(with the space), then use `Console.WriteLine` to print `World!`.

## Method Signature

```cs
public static void PrintGreeting()
```

## Expected Output

```cs
Hello, World!
```

## Hints

💡 1. Use `Console.Write("Hello, ")` to print the first part without a newline

💡 2. Follow it with `Console.WriteLine("World!")` to complete the line

💡 3. Remember: `Write` keeps the cursor on the same line, `WriteLine` moves to a new line after
