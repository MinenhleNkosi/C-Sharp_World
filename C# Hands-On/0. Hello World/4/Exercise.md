# Your Task

The `Calculate` method adds two numbers together and uses `return` to send back the result. Add a proper multi-line comment above the method (outside, not inside) that documents:

- What the method does
- What the parameters are
- What it returns

The method already works correctly - you just need to document it!

## Method Signature

```cs
public static int Calculate(int a, int b)
```

## Expected Results

```cs
Calculate(2, 3) -> 5
Calculate(0, 0) -> 0
Calculate(-1, 1) -> 0
```

## Hints

💡 1. Multi-line comments start with `/*` and end with `*/`

💡 2. Place the comment above the method declaration, outside the method body

💡 3. Good documentation explains WHAT the code does, not HOW it does it

💡 4. The `return` keyword sends a value back to whoever called the method
